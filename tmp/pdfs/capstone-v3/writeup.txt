Knowledge  Type  Functional  Purpose  Runtime  Implementation  
Procedural  Knowledge  Defines  how  to  reason  (identifying  prior  updates,  building  evidence  chains,  and  gating  choices).  
Hardcoded  in  the  AGENTS.md  harness,  Quality  Gates,  and  routing  states.

Dimension  Standard  Summarizer  BoardroomVoiceAgent  
Primary  Question  "What  does  this  document  say?"

Information  Flow  Passive  compression  (chronological).  
Logical  restructuring
