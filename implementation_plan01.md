# Goal Description

Build Wave 1 of the BoardroomVoiceAgent, an Enterprise Agent that converts executive materials into decision-grade spoken briefings using deterministic heuristics without LLM API integration. This agent will serve as a pre-decision clarity layer by extracting evidence, identifying decision gravity, surfacing missing evidence, and outputting a TTS-ready script following the Boardroom Handoff Protocol (AGENTS.md) and script_tts_ready_skill_v3_0.md.

## User Review Required

Please review the proposed deterministic heuristic approaches for each pipeline step. The logic relies on simple keyword/pattern matching and rule-based extraction since no LLMs are allowed in Wave 1.

## Open Questions

None at this time. The Wave 1 constraints (no LLMs, use plain Python, deterministic heuristics) are clear.
